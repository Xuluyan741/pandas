# 小熊猫智能管家 · 功能说明与使用指南

> 面向用户的产品功能说明与操作步骤，基于当前已实现能力整理。

---

## 一、项目是什么

**小熊猫智能管家（Super Project Agent）** 是一个以**对话为中心**的日程与任务管理应用，面向一人创业者、自由职业者，核心价值是：

- **认知减负**：主界面只有一个对话框，所有「项目管理、甘特图、任务列表」都收在右下角悬浮工作台里，不刷屏。
- **决策减负**：你说一句话（或语音），小熊猫帮你解析成任务、检测时间冲突、给出调整建议，你只需点「接受建议」或「忽略冲突」。
- **长期目标兜底**：说出「五一去韩国旅游」「备考雅思」等目标，小熊猫会搜资料、生成分步计划、每日推送监督，并支持一键写入日历。

技术形态是 **Web 应用**（Next.js），支持登录后数据同步到服务器（Turso），未登录时为演示模式（本地示例数据）。

## 二、功能总览

| 模块 | 功能说明 |
|------|----------|
| **主界面 · 对话** | 居中对话框，文字输入 + 语音输入，与小熊猫对话即可添加日程、处理冲突、规划长期目标。 |
| **冲突消解** | 新任务与已有任务时间重叠时，自动检测并给出「移动 / 顺延 / 取消」建议，用户确认后再写入。 |
| **Deep Link 执行器** | 识别打车、订餐、买票、会议、购物等意图，生成「预填链接」卡片，用户点「允许并打开」后跳转外部 App/网页，不自动下单或支付。 |
| **长期目标管家** | 识别考试/健身/项目/旅行等目标 → 用 Serper 搜索 + LLM 摘要资料 → LLM 生成分日计划（含资料链接）→ 一键写入日历；工作台可查看进度、暂停/恢复/完成目标。 |
| **Agent 自主推送** | 定时任务根据「逾期、今日到期、连续忙碌、成就、长期目标当日任务」等场景，生成 1 条推送（含长期目标监督与鼓励语）。 |
| **工作台（悬浮球）** | 大盘、任务列表、日历、甘特图、添加项目/WBS/单条任务、长期目标管理，全部收纳在右侧滑出面板。 |
| **配额管理** | 免费用户有每日次数限制（解析、对话、冲突建议等），耗尽后提示升级；Pro 用户更高额度。 |

## 三、使用方法

### 3.1 启动与访问

1. **本地运行**
   ```bash
   cd /Users/xuluyan/Documents/super-project-agent
   pnpm install   # 或 npm install
   pnpm dev       # 或 npm run dev
   ```
   浏览器打开 **http://localhost:3000**（或终端显示的端口）。

2. **环境变量（必选才能用 AI）**
   - 复制 `.env.local.example` 为 `.env.local`。
   - 至少配置：
     - `DEEPSEEK_API_KEY`：DeepSeek API Key（[平台](https://platform.deepseek.com/api_keys)），用于自然语言解析、冲突建议文案、长期目标规划。
   - 长期目标「搜资料」与**对话内网页搜索**建议再配置（二选一即可）：
     - `SERPER_API_KEY`：Serper 搜索 API Key（长期目标搜资料 + 对话里说「搜一下/查资料」时联网搜索）；不配则长期目标回退为搜索引擎链接，对话内搜索会提示未配置。
     - `BRAVE_API_KEY`：Brave 搜索 API，可替代 Serper 用于上述搜索。
   - 登录与数据库（可选）：
     - `NEXTAUTH_SECRET`、`NEXTAUTH_URL`；若用 Google 登录则配 `GOOGLE_CLIENT_ID`、`GOOGLE_CLIENT_SECRET`。
     - 生产环境配 `TURSO_DATABASE_URL`、`TURSO_AUTH_TOKEN`。
   - 推送（可选）：
     - `NEXT_PUBLIC_VAPID_PUBLIC_KEY`、`VAPID_PRIVATE_KEY`；定时任务需 `CRON_SECRET`。

3. **演示模式**
   - 未登录时进入即为演示模式：使用内置示例项目和任务，AI 对话仍会消耗配额（按 guest 计）。顶部有「演示模式」横幅，可点「登录」跳转。

### 3.2 主界面：和 small熊猫对话

- **输入框**：输入自然语言，例如：
  - 「明天下午三点和产品开会」
  - 「帮我规划五一去韩国旅游」
  - 「下周一下午 2 点高铁去上海」
- **发送**：点击发送或回车，小熊猫会：
  1. 流式显示「正在理解…」「正在检查冲突…」等状态（桌面端右下角有小熊猫状态面板）。
  2. 返回解析出的任务 + 若有冲突则附带建议文案。
  3. 若是长期目标（旅游、考试、健身、项目等），会先走「搜资料 + 生成计划」再返回清单和「一键写入到日历」按钮。
  4. 若识别到打车/订餐/买票等，会多一张「执行预览卡」，可点「允许并打开」跳转。
  5. 若说「搜一下 XXX」「查资料」「调研一下」等，会联网搜索并在回复中追加「【网页搜索】」结果（需配置 `SERPER_API_KEY` 或 `BRAVE_API_KEY`）。

- **冲突时怎么用**
  - 若出现「⚠️ 检测到日程冲突」和几条建议（如「移动『XXX』→ 3月2日 15:00」）：
    - **接受建议**：按建议调整已有任务时间并创建新任务。
    - **忽略冲突**：不改已有任务，直接创建新任务。

- **长期目标怎么用**
  - 输入如「帮我规划五一去韩国旅游」「备考雅思到 6 月」：
  - 小熊猫会返回一张「长期目标」卡片：分步计划（含日期、优先级、部分带资料链接）+ 推荐资料列表。
  - 点击 **「一键写入到日历」**：会创建一个对应「长期目标」实体，并把所有子任务写入当前日程（子任务带 `parentGoalId`，任务列表里会标「长期目标」和资料链接）。
  - 之后在 **工作台 → 大盘** 里可以看到「长期目标管家」区块：进度条、剩余天数、今日任务、**暂停 / 恢复 / 完成 / 移除** 按钮；健身类会带「仅供参考，请咨询专业人士」提示。

- **语音输入**
  - 点击输入框旁的麦克风，允许录音后说话，松开发送；后端会先转文字再走同一套解析与冲突逻辑（需配置 Whisper 相关环境变量才能用云端转写）。

---

### 3.3 工作台（悬浮球）

- 点击右下角 **悬浮球** 打开右侧滑出面板。
- **Tab：大盘**
  - 项目集分类（创业/工作/生活）、各项目进度、今日到期与逾期概览。
  - **长期目标管家**：当前所有长期目标的卡片（进度、剩余天数、今日任务、资料链接、暂停/恢复/完成/移除）。
  - AI 破局建议（今日最该做的一件事等）。
- **Tab：任务**
  - 任务列表（可筛 To Do / Doing / Done），每条可编辑、改状态、删除；带 `parentGoalId` 的会标「长期目标」并有「参考资料」链接。
- **Tab：日历**
  - 日历视图查看任务分布。
- **Tab：甘特图**
  - 多轨甘特图（依赖 Frappe Gantt CDN）。
- **Tab：添加**
  - 新建项目、WBS 批量录入、添加单条任务；底部有「快速添加任务」入口。

---

### 3.4 推送（需配置 Web Push + Cron）

- 用户订阅 Web Push 后，服务端定时（如每日）调用 `/api/cron/agent-push`（需带 `CRON_SECRET`）。
- 推送逻辑（`agent-push.ts`）：逾期提醒、今日到期、高优先级待办、连续忙碌关怀、成就鼓励、深夜关怀、长期目标当日任务 + 完成率鼓励/温和催促等，最多选 1 条发出；文案可经 LLM 润色。

---

### 3.5 配额与登录

- **免费用户**：每日 AI 解析/对话/冲突建议等有次数上限，用完后会提示「小熊猫今天的精力已经用完啦…」。
- **Pro 用户**：更高日/月额度（依赖用户表里 `subscription_status`、`subscription_plan` 等）。
- 登录后数据从服务器加载（`loadFromServer`），未登录为演示数据；配额按 `userId` 或 `guest` 统计。

## 五、常见问题

- **没有反应或报错「DeepSeek API key 未配置」**  
  检查 `.env.local` 中 `DEEPSEEK_API_KEY` 是否填写并重启 dev server。

- **小熊猫只回复「先休息」之类，没有把任务写入日程？**  
  若你说的是「明天晚上要完成 PPT」这类带时间 + 要完成的事，小熊猫应解析出一条任务并展示「接受/忽略冲突」卡片，点「接受」后才会真正写入日历。若仍只回复不出现任务卡片，请再试一次或说清楚时间与事项（如「帮我记一下：明天晚上要完成 PPT」）；我们已加强解析规则与 fallback，会尽量补一条任务。

- **怎么让 AI 调用 skill 或 MCP 帮我做 PPT？**  
  主界面对话目前**不会**把 MCP/技能当作可调用的工具交给模型，因此说「帮我做 PPT」只会得到文字建议或日程解析，不会真正执行「生成 PPT」的工具。若需调用 MCP 做 PPT：需配置 `MCP_SERVER_URL` 指向提供 PPT/文档生成能力的 MCP 服务，并调用 **POST /api/agent/mcp/chat**（或等后续在 UI 增加「MCP 助手」入口）。详见 `docs/MCP.md`。

- **长期目标没有推荐资料或只有一条搜索链接**  
  配置 `SERPER_API_KEY` 后，会先走 Serper 搜索再让 LLM 筛选摘要；未配置则只回退为一条搜索引擎链接。

- **推送收不到**  
  确认已配置 VAPID 密钥、用户已订阅推送、定时任务带 `CRON_SECRET` 调用 `/api/cron/agent-push`。

- **登录后数据是空的**  
  确认 `TURSO_*` 或本地数据库正常，且 `loadFromServer` 调用的 `/api/projects`、`/api/tasks` 返回正常。
